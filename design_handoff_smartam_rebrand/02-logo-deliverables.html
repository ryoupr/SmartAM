06INSTALLATION GUIDE

SmartAM プロジェクトへの組み込み手順

  1. 上記の PNG ボタンを順番にクリック → 全サイズダウンロード
  2. ダウンロードした PNG を app/src-tauri/icons/ 配下の同名ファイルに上書き
    • icon.png (1024 → 1024)
    • 128x128.png / 128x128@2x.png / 32x32.png
    • Square*Logo.png は Windows 用なので、しばらく未使用なら後回しで OK
  3. favicon.svg を app/src/lib/assets/favicon.svg に上書き（現在 Svelte の初期テンプレート）
  4. icon.icns は iconutil -c icns icon.iconset で再生成（macOS）。次のターンでコマンドを案内可能
  5. app/src-tauri/tauri.conf.json の bundle.icon パスは現状のまま使い回し可
